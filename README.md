# 🧪 Analysis of Chemical Components

## 📌 Project Overview

This project analyzes the chemical ingredients of cosmetic products and identifies products with similar ingredient compositions. A content-based recommendation approach is used to compare moisturizers suitable for dry skin.

The project processes ingredient lists using Natural Language Processing (NLP), converts them into numerical vectors using One-Hot Encoding, reduces dimensionality with t-SNE, and visualizes ingredient similarity using Bokeh.

---

## 🎯 Project Objective

- Analyze cosmetic ingredient data
- Filter moisturizers suitable for dry skin
- Create an ingredient vocabulary
- Build a Document-Term Matrix
- Visualize product similarity using t-SNE
- Compare products based on ingredient composition

---

## 📂 Dataset

- Source: Sephora Cosmetics Dataset
- Total Products: 1472
- File: `cosmetics.csv`

## 📋 Project Workflow

3. Tokenize Ingredients
4. Build Ingredient Dictionary
5. One-Hot Encoding
6. Create Document-Term Matrix
7. Apply t-SNE
8. Visualize Products
9. Compare Similar Products
